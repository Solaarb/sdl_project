# 🐘 Elephant Dodge – Jeu en C avec SDL
Elephant Dodge est un jeu d'arcade en 2D développé en C avec la bibliothèque SDL2. Vous incarnez un éléphant courageux qui doit esquiver des palmiers et d'autres éléphants dans une course effrénée pour survivre le plus longtemps possible !

## 🕹️ Contrôles
**←↑↓→/ ZQSD** : Déplacement de l’éléphant\
**Echap** : Quitter le jeu  

## 🧰 Technologies utilisées
**Langage** : C\
**Librairie** : SDL2 (Simple DirectMedia Layer)\
**Compatibilité** : Windows, Linux, macOS (avec SDL installée)

## 🔧 Installation
### Prérequis
SDL2 doit être installée sur votre machine.
Sous Debian / Ubuntu :
```
sudo apt-get install libsdl2-dev
```
Sous macOS avec Homebrew :
```
brew install sdl2
```

### Compilation
Dans le terminal, exécutez :
```
make 
```
Lancement du jeu
```
./jeu.exe
```
